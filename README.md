## Features:  
### Seed based generation:
-> generates same terrain if same seed was entered. Supports all unicode characters (even emojies).
### Layers details strenght slider 
-> allows to change strenght of layers (max heigh) from -60 to 60, allowing to get various genrations from same seed  
some base generations can be boring, but light adjust of slider gives outstanding results.

### Gradient:
  Gradient slider allows to create gradient to paint the terrain, user can add and remove nodes by clicking **"+"** and **"-"**  
  Strange lines and circle below allows user to change color of nodes, they will become active after click on any node.
